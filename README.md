Mr_Expence's Pvp Overlay changes some textures to be easier to see or hear. This resource pack was inspired from [TheobaldTheBird's Bare Bones PVP Texture Pack](https://www.youtube.com/watch?v=gVsjkErSXeg) and some other texture packs. 

- Blue Cobwebs
- Shorter Swords
- Breach and Density Identifiable Maces
- Better Totem Sound
- Low Fire
- Better Effect Particles
- Bare Bone Buttons
